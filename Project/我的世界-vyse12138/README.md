![preview](https://user-images.githubusercontent.com/88306344/154383952-9b33bad4-eebb-4a98-a12e-f5f137422d06.gif)

## 已支持:

- 方块的放置 / 破坏
- 鼠标滚轮（数字键）选择不同的方块类型
- 移动和碰撞检测
- 随机的地形和树木生成
- 无限的世界
- 保存 / 读取游戏
- 音效和背景音乐
- 可调节的渲染距离和视野范围
- 高亮准心方块
- 基本的 UI

## 开发中:

- 生成水

- 按住鼠标来进行方块的放置 / 破坏
- 更多的保存栏位
- 更高效的算法
